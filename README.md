# نظام إدارة مركز تدريب مع إدارة دورات وشهادات
==========================

## ملخص المشروع
----------------

نظام إدارة مركز تدريب مع إدارة دورات وشهادات هو تطبيق إلكتروني يهدف إلى تسهيل إدارة المراكز التدريبية من خلال إدارة دورات التدريب وشهادات التخرج. يتيح هذا النظام للمستخدمين إدارة الدورات التدريبية، وإضافة وتعديل بيانات الدورات، وإصدار شهادات التخرج، وتتبع أداء المركز التدريبي.

## هيكل المشروع
----------------

* `app/`: هيكل المشروع الرئيسي
 + `__init__.py`: ملف تعريف المشروع
 + `models/`: هيكل البيانات
 + `routes/`: هيكل الطرق
 + `services/`: هيكل الخدمات
* `requirements.txt`: قائمة المتطلبات
* `docker-compose.yml`: ملف تعريف docker-compose
* `Dockerfile`: ملف تعريف docker

## إعداد البيئة باستخدام docker-compose
--------------------------------------

### خطوات الإعداد

1. تأكد من وجود docker و docker-compose على جهازك.
2. افتح المجلد الرئيسي للمشروع في terminal.
3. استخدم الأمر `docker-compose up` لتشغيل البيئة.
4. افتح متصفحك ومرر إلى `http://localhost:5000` لفتح تطبيق النظام.

### تفاصيل الإعداد

* `docker-compose.yml` يحدد البيئة التي ستتشغيل فيها التطبيق.
* `Dockerfile` يحدد كيفية بناء الصورة docker.
* `requirements.txt` يحدد المتطلبات التي يجب توافرها في البيئة.

## وحدات المشروع
----------------

### وحدات البيانات

* `دورات التدريب`: يحتوي على بيانات الدورات التدريبية.
* `شهادات التخرج`: يحتوي على بيانات شهادات التخرج.
* `المستخدمين`: يحتوي على بيانات المستخدمين.

### وحدات الطرق

* `دورات التدريب`: يتيح إضافة وتعديل بيانات الدورات التدريبية.
* `شهادات التخرج`: يتيح إصدار شهادات التخرج.
* `المستخدمين`: يتيح إضافة وتعديل بيانات المستخدمين.

### أدوار المستخدمين

* `المدير`: يمكنه إدارة جميع الوحدات.
* `المشرف`: يمكنه إدارة الوحدات المتعلقة بالدورات التدريبية.
* `المستخدم العادي`: يمكنه إدارة الوحدات المتعلقة بالدورات التدريبية.
